Workshop

1. Write a function that takes an array of strings as input and returns a new array with all strings converted to uppercase.

  • Input Test Cases: ["hello", "world", "abc"]
  • Output Test Cases: ["HELLO", "WORLD", "ABC"]
Uppercase

2. Write a function that takes an array of strings as input and returns a new array with the first 3 characters of each string.

  • Input Test Cases: ["hello", "world", "abc"]
  • Output Test Cases: ["hel", "wor", "abc"]
Substring

3. Write a function that takes an array of strings as input and returns a new array with all occurrences of "o" replaced with "0".

  • Input Test Cases: ["hello", "world", "abc"]
  • Output Test Cases: ["hell0", "w0rld", "abc"]
Replace

4. Write a function that takes an array of strings as input and returns a new array with each string split into words.

  • Input Test Cases: ["hello-world", "abc-def"]
  • Output Test Cases: ["hello", "world", "abc", "def"]
Split

5. Write a function that takes an array of strings as input and returns a new array with only the strings longer than 5 characters.

  • Input Test Cases: ["hello", "world", "abc", "defghi"]
  • Output Test Cases: ["defghi"]
Filter

6. Write a function that takes an array of strings as input and returns a new array with all strings converted to uppercase and "!" appended.

  • Input Test Cases: ["hello", "world", "abc"]
  • Output Test Cases: ["HELLO!", "WORLD!", "ABC!"]
Map

7. Write a function that takes an array of strings as input and returns the first string containing "o".

  • Input Test Cases: ["hello", "world", "abc", "defghi"]
  • Output Test Cases: "world"
Find

8. Write a function that takes an array of strings as input and returns a new string with all strings joined by commas.

  • Input Test Cases: ["hello", "world", "abc"]
  • Output Test Cases: "hello,world,abc"
Join
